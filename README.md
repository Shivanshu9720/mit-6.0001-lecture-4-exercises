# mit-6.0001-lecture-4-exercises

Solutions to all coding exercises, projects, and homework assignments covered in Lecture 4 of MIT's 6.0001 Introduction to Computer Science course.

### Overview
* All projects use the **GUESS AND CHECK** method.
* In this method, we check for the correct value in a sequence one by one.
* **GUESS AND CHECK** is a simple algorithm for solving problems.
* It is efficient for small values, but it can make the code slow for larger values.

---

### 1. PROJECT 1 (lec4_project1_sum.py)
* Implemented a script about the summation of numbers in a given range.

### 2. PROJECT 2 (lec4_project2_even_counter.py)
* Implemented a Python script that counts the total number of even numbers within a specific range using a for-loop and the modulo operator.

### 3. PROJECT 3 (lec4_project3_unique_letters.py)
* Implemented a script to count and extract unique letters from a string using the 'not in' operator.
* Included comments explaining a previous logic mistake regarding string comparisons for learning and revision purposes.

### 4. PROJECT 4 (lec4_project4.py)
* Built using a while loop for sequential guessing.
* Checks if the user input is a perfect square.
* Includes basic validation warning for negative integers.
